## Node version

    v12.22.12

## first time?

    1. download nvm/node
    2. make sure node version is v12.22.12 (`node -v`)
    3. install dependencies (`npm i`)

## 開發

    `npm run dev`

_存檔時會自動編譯檔案_

## 建立最小化的 CSS

    `npm run build`

_最小化 css_
